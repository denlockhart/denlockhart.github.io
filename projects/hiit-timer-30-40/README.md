# HIIT Timer 30/40

Simple interval timer: **30 seconds work**, **40 seconds rest**, repeat for a set number of rounds.

- **Live:** https://www.denlockhart.com/projects/hiit-timer-30-40/
- **Folder:** `projects/hiit-timer-30-40/`
- **Also see:** [HIIT Timer 20/30](../hiit-timer/)

## Features

- Fixed 30s / 40s intervals
- Configurable round count (saved in `localStorage`)
- Optional beep on phase changes
- Pause / resume / reset

## Local development

```bash
npm start
# http://localhost:3000/projects/hiit-timer-30-40/
```
